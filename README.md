# Spaces → Dots generator
This simple generator instantly converts any pasted text by removing all spaces and replacing them with dots. Just paste your content into the input box, click Convert, and the cleaned version appears below. It’s designed for fast, frictionless use—perfect for formatting filenames, tags, identifiers, or any text that needs compact dot‑separated output.

#
### HOW DOES IT WORK?
It uses JS and HTML to remove spaces and add DOTS.
